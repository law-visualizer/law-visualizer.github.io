    Connections

    Ticks represent titles, chapters, or sections depending on zoom level. Arcs connect units that share an issue; dots mark issues confined to a single unit. To zoom in, click a label below the tick (or click a bar in “By Title” below). Use the breadcrumb to zoom out.

    About

    This site is a proof-of-concept mapping of issues in NH Revised Statutes. The full text of every cited section is fetched from the official gc.nh.gov site.

    Issues are curated manually — automated language models miss nuance. Each issue entry includes the exact quoted text, a category, and a link back to the official source.

    Source code and data available in the project repository.

    Data: NH Revised Statutes via gc.nh.gov. Non-commercial, educational use.
